# Vertical Caps Holder (180mm diameter)

Параметрический держатель для вертикальных крышек диаметром 180 мм. Держатель с двумя цилиндрическими вырезами, проходящими вдоль оси X, для размещения крышек в вертикальном положении.

- Файл модели: `caps-holder.scad`
- Версия: 1.0
- Оси: X — продольная ось (вдоль цилиндров), Y — поперечная ось, Z — вертикальная ось

## Состав модели (модули/фрагменты)
- `base` — полная сборка держателя (корпус с цилиндрическими вырезами)
- `deck_core` — основной корпус (прямоугольный блок)
- `end_stop` — торцевые упоры по X (слева и справа)
- `cylinder_cutter` — цилиндрический вырез для крышки (вдоль оси X)
- `cradle_cuts` — объединение всех выемок
- `clip_for_fragments` — клиппер для режима тест‑фрагментов

## Ключевые параметры
```scad
// Точность
$fn = 0; $fa = 6; $fs = 0.35; pin_fs = 0.25;

// Тест‑фрагменты
test_fragment = false;      // true — печатать только фрагменты
frag_size     = 20;         // окно вырезки, мм
frag_gap_x    = 10;         // разнос фрагментов, мм
frag_h_extra  = 12;         // запас по высоте, мм

// Габариты крышки
cap_diam      = 180;        // диаметр крышки, мм
cap_width     = 20;         // ширина крышки по X, мм
num_caps      = 2;          // количество крышек

// Геометрия держателя
wall_th       = 1;          // толщина стенок, мм
holder_length = 180;        // длина держателя по Y, мм
end_stop_th   = 1;          // толщина торцевых упоров по X, мм
pad_between   = 1;          // зазор между крышками по X, мм

// Ограничение по высоте
max_h         = 80;         // максимум по Z, мм
```

## Идея конструкции
- Прямоугольный корпус с двумя цилиндрическими вырезами, проходящими вдоль оси X
- Каждый цилиндр имеет диаметр `cap_diam` (180 мм) и длину `cap_width` (20 мм) по оси X
- Цилиндры подняты на радиус по Z для правильного размещения вертикальных крышек
- Торцевые упоры по X предотвращают смещение крышек
- Высота модели ограничена параметром `max_h`

## Печать
- Ориентация: печатать как есть, Z вверх. Корпус образует базу, цилиндрические вырезы сверху.
- Поддержки не требуются.
- При необходимости включайте `test_fragment=true` для проверки посадки по диаметру/зазору — будет выведен укороченный клип.

## Использование параметров
- Для других крышек меняйте `cap_diam`, `cap_width`, `num_caps`, `pad_between`
- `holder_length` определяет максимальную длину по Y
- `max_h` ограничивает высоту модели и срезает верхнюю часть цилиндров
- Толщина стенок регулируется параметром `wall_th`

## План SCAD‑файла
- Версия модели и комментарии
- Переменные (все размеры вверху файла)
- Модули фрагментов (`deck_core`, `end_stop`, `cylinder_cutter`, `cradle_cuts`)
- Сборка `base()` и вывод через `clip_for_fragments()`

# lid-dispenser

Коллекция крышек для бутылок с воронками для дозаторов.

## Модели

### lid-38-400
- Файл модели: `lid-38-400.scad`
- Версия: 1.0
- Описание: Стандартная крышка 38-400 (GL38 class) с резьбой

### lid-38-400-to-funnel-20
- Файл модели: `lid-38-400-to-funnel-20.scad`
- Версия: 1.0
- Описание: Крышка 38-400 с воронкой для дозатора

## Фрагменты модели lid-38-400-to-funnel-20

- `cap` — крышка 38-400 с резьбой (BOSL2 `sp_cap`), параметры: `diam`, `type`, `wall`, `slop`
- `funnel` — воронка с большим верхним диаметром и узким выходом, параметры: `funnel_top_diam`, `funnel_bottom_diam`, `funnel_length`, `funnel_wall`
- Главная деталь: `show_all()` = `cap + funnel`

## Параметры lid-38-400-to-funnel-20

### Крышка
- `diam` — номинальный диаметр резьбы, 38мм (SP38-400)
- `type` — тип резьбы, 400
- `wall` — толщина стенки крышки, 2.2мм
- `slop` — дополнительный зазор под принтер/пластик, 0.20мм

### Воронка
- `funnel_top_diam` — верхний диаметр воронки (больше крышки), 50мм
- `funnel_bottom_diam` — нижний диаметр воронки (выход), 20мм
- `funnel_length` — длина воронки, 30мм
- `funnel_wall` — толщина стенки воронки, 2.0мм

## Технические настройки

- `$fn=0`, `$fa=6`, `$fs=0.35`, `pin_fs=0.25`
- Использует BOSL2 библиотеку для создания стандартных крышек

# Подставка для ароматической палочки

- Файл модели: `incense-holder.scad`
- Версия: 1.0

## Описание
Подставка для ароматической палочки с расширяющимся основанием с вогнутыми стенками для устойчивости и вогнутой полусферой вверху для сбора пепла. Состоит из трёх частей: основания, ножки и чаши для пепла.

## Фрагменты
- `base` — расширяющееся основание с вогнутыми стенками (усечённый конус) для устойчивости, диаметр 40 мм снизу, 4 мм сверху
- `stem` — ножка (полый цилиндр) с внешним диаметром 4 мм и внутренним 1.8 мм
- `bowl` — перевёрнутая вогнутая полусфера для сбора пепла, диаметр 50 мм, обрезана на 1/3 сверху

## Ключевые параметры (см. начало SCAD)
- $fn, $fa, $fs, pin_fs — точность окружностей
- test_fragment, frag_* — тест‑фрагменты
- edge_chamfer_*, tiny — фаски/совм.
- `stem_height` — высота ножки (20 мм)
- `stem_outer_d` — внешний диаметр ножки (4 мм)
- `stem_inner_d` — внутренний диаметр ножки (1.8 мм, отверстие для палочки)
- `base_bottom_d` — диаметр основания снизу (40 мм)
- `base_top_d` — диаметр основания сверху (4 мм, равен диаметру ножки)
- `base_height` — высота основания (4 мм)
- `base_concave_depth` — глубина вогнутости стенок основания (2 мм)
- `bowl_diameter` — диаметр сферы для чаши (50 мм)
- `bowl_wall_thickness` — толщина стенок чаши (1 мм)

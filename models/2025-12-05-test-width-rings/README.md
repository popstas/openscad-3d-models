# test width rings

- Файл модели: `test-width-rings.scad`
- Версия: 1.0

## Описание

Набор тестовых колец для проверки толщины стенок при 3D печати. Каждое кольцо имеет свою толщину стенки (1, 1.6, 2 мм) с текстовой меткой на боковой поверхности (XZ плоскость).

## Фрагменты модели

- `ring_1mm` — кольцо с толщиной стенки 1 мм
- `ring_1_6mm` — кольцо с толщиной стенки 1.6 мм
- `ring_2mm` — кольцо с толщиной стенки 2 мм
- `text_1mm` — текстовая метка "1" на кольце 1 мм
- `text_1_6mm` — текстовая метка "1.6" на кольце 1.6 мм
- `text_2mm` — текстовая метка "2" на кольце 2 мм

Главная деталь: `show_all()` = все кольца с текстовыми метками, расположенные с промежутком 5 мм

## Параметры модели

### Кольца
- `ring_diam` — внешний диаметр кольца, 20 мм
- `ring_height` — высота кольца, 10 мм
- `ring_thicknesses` — массив толщин стенок, [1, 1.6, 2] мм
- `ring_spacing` — расстояние между кольцами по оси X, 5 мм

### Текст
- `text_size` — размер шрифта, 4 мм
- `text_height` — высота текста (толщина выдавливания), 0.5 мм

## Ключевые параметры (см. начало SCAD)
- $fn, $fa, $fs, pin_fs — точность окружностей
- test_fragment, frag_* — тест‑фрагменты
- edge_chamfer_*, tiny — фаски/совм.
